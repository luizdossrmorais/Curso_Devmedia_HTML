AULA6 - FORMULARIO.

Elementos/Controles de Formulario:

** = No elemento label escrevemos o titulo que vai referenciar ao input posterior.

** = O elemento input como padrão é um campo do tipo texto (type="text"), porem ele pode se transformar no que quizermos (email, date, password, etc) se caracterizando assim como totalmente modelavel.

Atributos de Formulario:

** No Elemento label:
[for: recebe o valor no id do input.]
** No Elemento input:
[type: podemos informar varios tipor de dados, aqui que modificamos a forma do input a nossa preferencia.]
[placeholder: podemos digitar um texto para aparecer no campo ainda vazio de input, assim mostrando ao usuario que tipo de dado, e como se deve preencher aquele campo, quando o usuario lica no campo, esse texto some.]
[min/max: define o valor minimo e valor maximo em um campo do tipo numero (type="number").]
**No Elemento select:
[value: Adicionamos um valor para o usuario selecionar no input do tipo select (type="radio").]
[name: Em um campo do typo select (type="radio"), o name agrupa valores, e faz o controle para que um valor apenas seja selecionado. (ex: sexo)]
[checked: Atributo que seleciona 1 campo do tipo radio, mesmo contendo 2, por isso é preciso colocar esse atributo em apenas um campo. (ex: sexo)]
